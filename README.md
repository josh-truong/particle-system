# particle-system

## Required Submodules
- ### glfw
  ```
    git rm -rf external/glfw
    git submodule add https://github.com/glfw/glfw.git external/glfw
  ```
- ### glm
  ```
    git rm -rf external/glm
    git submodule add https://github.com/g-truc/glm.git external/glm
  ```

---
### Configure
```
    sh ./configure.sh
```

### Build
```
    sh ./build.sh
```

### Run
```
    sh ./run.sh
```
